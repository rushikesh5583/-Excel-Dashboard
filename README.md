# Excel-Dashboard project
Interactive Sales and Profit Dashboard built in Excel to track key business metrics, including sales, profit, profit margin, and regional performance. Features dynamic charts, pivot tables, and slicers for data analysis, trend identification, and informed decision-making.

# Key Features
Total Sales and Total Profit KPIs
Monthly and Yearly Sales Trends
Profit Margin Analysis
Regional Sales Performance
Product Category Analysis
Top Customers and Best-Selling Products
Interactive Filters (Slicers) for Dynamic Reporting
Visual Charts and Graphs for Easy Insights
